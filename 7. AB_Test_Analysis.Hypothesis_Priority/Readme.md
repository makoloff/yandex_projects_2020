Приоритизация гипотез. Анализ A/B теста.
Описание проекта
Вы — аналитик крупного интернет-магазина. Вместе с отделом маркетинга вы подготовили список гипотез для увеличения выручки.
Приоритизируйте гипотезы, запустите A/B-тест и проанализируйте результаты.

В данном исследовании представлены 3 файла с данными:
1-й - таблица с параметрами для каждой из 9 гипотез.
2-й - таблица с заказами для обеих тестируемых групп, перечислены дата заказа, id заказа, id покупателя, выручка с заказа.
3-й - таблица о посещении сайта для обеих групп, перечислены дата посещения, кол-во посетителей на конкретную дату.
