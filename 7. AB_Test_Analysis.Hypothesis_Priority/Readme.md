# Приоритизация гипотез. Анализ A/B теста.

Проведена приоритизация гипотез по фреймворкам ICE и RICE. Затем проведен анализ
результатов A/B-теста, построены графики кумулятивной выручки, среднего чека,
конверсии по группам, а затем посчитана статистическая значимость различий конверсий
и средних чеков по сырым и очищенным данным. На основании анализа мной было
принято решение о нецелесообразности дальнейшего проведения теста.

Задача:
приоритизировать гипотезы по 2-м фреймворкам (ICE\RICE)
проанализировать результаты A/B теста на предмет увеличения выручки


## Примененные методы и библиотеки в проекте
- расчет приоритизации гипотез по фреймворкам ICE и RICE
- проверка гипотез о нормальности распределения посредством критерия Шапиро-Уилка
- проверка гипотез "сдвига" посредством непараметрического критерия Манна-Уитни
- библиотека визуализации Matplotlib
- библиотека SciPy
