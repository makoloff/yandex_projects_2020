# Определение перспективного тарифа для телеком компании
Проведен предварительный анализ использования тарифов на выборке клиентов,
проанализировано поведение клиентов при использовании услуг оператора и
рекомендованы оптимальные наборы услуг для пользователей. Проведена предобработка
данных, их анализ. Проверены гипотезы о различии выручки абонентов разных тарифов и
различии выручки абонентов из Москвы и других регионов.

Задача - определить наиболее перспективный тариф для компании с точки зрения выручки.

## Примененные методы и библиотеки в проекте
- визуализация посредством библиотеки Matplotlib, Plotly
- проверка статистических гипотез посредством t-критерия Стьюдента (t-test)
