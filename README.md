# Выполненные проекты в рамках обучения в Яндекс.Практикум

Описание к каждому проекту находится в файлах Readme, а также внутри каждого ipynb файла.

## В зависимости от проекта применены следующие библиотеки Python
- библиотеки для анализа данных pandas, numpy
- библиотеки визуализации plotly, seaborn, matplotlib
- библиотеки pymystem3, stats

##  и методы анализа данных
- корреляционный анализ
- когортный анализ
- проверка гипотез посредством параметрических критерия t-Стюдента
- проверка гипотез о нормальности распределения посредством критерия Шапиро-Уилка
- проверка гипотез посредством непараметрического критерия хи-квадрат
- проверка гипотез "сдвига" посредством непараметрического критерия Манна-Уитни
