# Мобильное приложение по продаже продуктов питания.

Были изучены принципы событийной аналитики. Я построил
воронку продаж, исследовал путь пользователей до покупки. Проанализировал
результаты A/B-теста введения новых шрифтов. Сравнил 2 контрольных группы между
собой, убедился в правильном разделении трафика, а затем сравнил с тестовой группой
Выявлено, что новый шрифт статистически значимо не повлиял на поведение пользователей.

Задача:
Анализ поведения пользователей в мобильном приложении по продаже продуктов питания. Изучить воронку продаж, выявить "проблемные" шаги до покупки. 
Проанализировать результаты А/А/В эксперимента.

## Примененные методы и библиотеки в проекте
- библиотеки визуализации Plotly, Matplotlib
- проверка гипотез посредством непараметрического критерия хи-квадрат
- поправка Бонферрони на множественное сравнение
