# Оптимизация маркетинговых затрат.

Вас пригласили на стажировку в отдел аналитики Яндекс.Афиши. Первое задание: помочь маркетологам оптимизировать маркетинговые затраты.
У вас в распоряжении есть данные от Яндекс.Афиши с июня 2017 по конец мая 2018 года:
лог сервера с данными о посещениях сайта Яндекс.Афиши, выгрузка всех заказов за этот период, статистика рекламных расходов.

Задача - определить какие источники/платформы более перспективны и актуальны для развития.

## Примененные методы и библиотеки в проекте
- когортный анализ
- библиотеки визуализации Seaborn, Plotly

## Рассчитанные продуктовые бизнес-метрики в проекте
- DAU\MAU\WAU
- Retention
- LTV

- ROMI
