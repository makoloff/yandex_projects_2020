# Создание дашборда по пользовательским событиям для агрегатора новостей

Задача:
Используя данные Яндекс.Дзена построить дашборд с метриками взаимодействия пользователей с карточками статей.

Выполненный порядок действий:

1. Подключение к сторонней базе данных
2. Выполнен запрос на необходимую таблицу с данными
3. Прочитана таблица
4. Сохранена таблица во внешний файл csv и на основе этих данных построен дашборд в системе Tableau Public
5. По результатам была подготовлена презентация с полученными графиками, выводами и рекомендациями

## Примененные методы в проекте
- создание связи и подключение к сторонней БД
- визуализации и построение дашборда в Tableau Public
